# Struct — Build Systems, Not To-Do Lists

Struct is a minimal accountability app designed to help users build discipline through **consistent daily execution**, not endless planning.

This is NOT a typical todo app or habit tracker.

---

## 🧠 Core Philosophy

Most productivity apps fail because they focus on:

* Too many features
* Too much flexibility
* Endless task creation

Struct focuses on:

* **Execution over planning**
* **Consistency over intensity**
* **Systems over random tasks**

---

## 🏗️ Core System

Struct is built around 3 layers:

### 1. Structures (Identity Layer)

* Users can create **maximum of 2 structures**
* A structure represents a life system

Examples:

* Study Daily
* Build Every Day
* No Porn

Structures are **fixed** and define how a user wants to live.

---

### 2. Habits (Inside Structures)

* Each structure contains multiple habits

Example:

Structure: Study Daily

* Read notes
* Practice problems
* Revise

---

### 3. Tasks (Execution Layer)

* Tasks are generated daily from habits
* Users **do not manually create tasks like a todo app**
* Tasks are completed daily and reset

---

## 📊 Feedback System

Struct measures discipline through:

### Daily Completion

* Example: 2/4 habits completed

### Streaks

* Based on consistent execution
* Continues only if user meets a threshold

### System Streak

* If 2 structures exist, both must pass
* System streak = minimum of structure streaks

---

## 🧩 Pages & Flow

* **Home** → Overview (structures, today's tasks, stats)
* **Tasks** → Execution mode (focus + grouped tasks)
* **Structures** → Define and manage life systems
* **History** → Daily performance logs
* **Stats** → Long-term insights and trends
* **Profile** → User settings and preferences
